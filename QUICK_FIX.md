# Быстрое решение: Бэкенд не доступен

## Проблема
Фронтенд показывает "Бэкенд не доступен", хотя бэкенд может быть запущен.

## Решение

### 1. Убедитесь, что бэкенд запущен

Откройте новое окно терминала и выполните:
```bash
start_backend.bat
```

Или вручную:
```bash
cd AdaptEd\backend
.\.venv\Scripts\activate
python -m uvicorn app:app --host 127.0.0.1 --port 8000 --reload
```

### 2. Проверьте, что бэкенд отвечает

Откройте в браузере: http://localhost:8000

Должно появиться:
```json
{"message": "Welcome to the AdaptEd API!"}
```

### 3. Обновите страницу фронтенда

1. Нажмите **F5** или **Ctrl+R** для обновления страницы
2. Подождите несколько секунд - статус должен обновиться автоматически
3. Если статус все еще "не доступен", нажмите кнопку **"Проверить снова"**

### 4. Проверьте консоль браузера

1. Нажмите **F12** для открытия DevTools
2. Перейдите на вкладку **Console**
3. Посмотрите на ошибки (если есть)
4. Перейдите на вкладку **Network**
5. Попробуйте войти снова
6. Проверьте запрос к `http://localhost:8000/` - какой статус?

## Новые возможности

После обновления фронтенда:
- ✅ Автоматическая проверка статуса бэкенда каждые 5 секунд
- ✅ Кнопка "Проверить снова" для ручной проверки
- ✅ Индикатор состояния: проверка/онлайн/офлайн
- ✅ Увеличенный таймаут для проверки (5 секунд)

## Если проблема сохраняется

1. **Проверьте, что бэкенд запущен на правильном порту:**
   ```bash
   netstat -ano | findstr :8000
   ```

2. **Проверьте, что нет конфликтов портов:**
   - Убедитесь, что другой процесс не использует порт 8000
   - Если нужно, измените порт в `start_backend.bat`

3. **Проверьте настройки CORS:**
   - В `AdaptEd/backend/app.py` должно быть: `allow_origins=["*"]`

4. **Очистите кэш браузера:**
   - Нажмите **Ctrl+Shift+Delete**
   - Очистите кэш и cookies
   - Или используйте режим инкогнито

## Тестирование подключения

Откройте консоль браузера (F12) и выполните:
```javascript
fetch('http://localhost:8000/')
  .then(r => r.json())
  .then(console.log)
  .catch(console.error)
```

Если видите `{"message": "Welcome to the AdaptEd API!"}`, то бэкенд доступен.
