# Руководство по авторизации

## Как войти в систему

### Вариант 1: Регистрация нового пользователя

1. Откройте приложение (http://localhost:3000)
2. Нажмите на вкладку "Регистрация"
3. Заполните форму:
   - **ФИО**: Ваше полное имя
   - **Email**: Ваш email адрес
   - **Пароль**: Минимум 6 символов
   - **Роль**: Выберите роль (Ученик/Учитель/Родитель)
   - **Класс** (для учеников): Опционально, например "7А"
   - **Телефон**: Опционально
4. Нажмите "Зарегистрироваться"
5. После регистрации вы автоматически войдете в систему

### Вариант 2: Вход существующего пользователя

1. Откройте приложение
2. На вкладке "Вход" введите:
   - **Email**: Ваш email
   - **Пароль**: Ваш пароль
3. Нажмите "Войти"

## Роли пользователей

- **Ученик** - доступ к личному кабинету, заданиям, прогрессу
- **Учитель** - доступ к аналитике, назначению заданий, отчетам
- **Родитель** - доступ к просмотру прогресса ребенка
- **Администратор** - полный доступ ко всем функциям

## Технические детали

- Токены авторизации хранятся в `localStorage`
- Токен автоматически добавляется к каждому API запросу
- При истечении токена пользователь автоматически разлогинивается
- Для выхода нажмите кнопку "Выход" в правом верхнем углу

## Устранение проблем

Если не можете войти:
1. Проверьте, что бэкенд запущен на порту 8000
2. Проверьте правильность email и пароля
3. Убедитесь, что пользователь зарегистрирован
4. Очистите localStorage в браузере (F12 -> Application -> Local Storage -> Clear)
