### Добавленные функции (Features)

1. **Расширенная система задач**
   - Добавлены 3 уровня сложности: beginner, intermediate, advanced
   - Всего 18 математических задач (по 6 на каждый уровень)
   - Система категоризации задач

2. **Новые API Endpoints**
   - `GET /tasks?difficulty={level}` - получение задач по уровню сложности
   - `GET /tasks/random?difficulty={level}` - получение случайной задачи
   - `GET /tasks/categories` - получение всех категорий задач
   - Улучшенный `POST /tasks/check` с расширенной обратной связью

3. **Улучшенный сервис рекомендаций**
   - Анализ производительности пользователя
   - Определение уровня сложности на основе результатов
   - Персонализированные рекомендации
   - Система статистики и трендов

4. **Скрипты для запуска**
   - `run_backend.py` - простой запуск backend сервера
   - `run_frontend.py` - простой запуск frontend приложения

5. **Документация**
   - Добавлен `.gitignore`
   - Создан `README_installation.md` с подробными инструкциями
   - Создан `CHANGELOG.md` (этот файл)

### Что нужно сделать для использования

1. Установить зависимости:
   ```bash
   cd AdaptEd/backend
   pip install -r requirements.txt
   cd ../frontend
   pip install -r requirements.txt
   ```

2. Запустить backend:
   ```bash
   cd AdaptEd
   python run_backend.py
   ```

3. Запустить frontend (в другом терминале):
   ```bash
   cd AdaptEd
   python run_frontend.py
   ```

4. Открыть браузер: http://localhost:8501

### Известные ограничения

- Данные хранятся в памяти (при перезапуске теряются)
- Пока доступен только один предмет (математика)
- Интерфейс на английском языке
- Нет системы аутентификации пользователей
